Arduino RP2040 RESTful Weather Server

Use an Arduino Nano to read real-time temperature and humidity data from a sensor. The readings are served as JSON through a RESTful API using the onboard Wi-Fi module. It includes serial monitoring for debugging and is designed to be easily consumed by JavaScript-based frontends.

Equipment used: Arduino Nano RP2040 Connect, AHT20 sensor

Tech used: Arduino, C++, WiFiNINA, ArduinoJson, JavaScript, HTML, CSS
